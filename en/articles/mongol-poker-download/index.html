What you need

  • Stable internet

Step 1 — Download PPPoker

Install from official sources only: app store or the official PPPoker site.

Android

  1. Open Google Play Store and search «PPPoker»
  2. Select the official PPPoker app
  3. Tap Install

iPhone / iPad

  1. Open App Store
  2. Search «PPPoker»
  3. Install and open

Windows or Mac

  1. Download the official PPPoker desktop client
  2. Follow the installer
  3. One account works on all devices

Many players prefer desktop for a larger screen; one PPPoker login is enough.

Step 2 — Create an account

  1. Tap Register / Sign up
  2. Enter username, email, password
  3. Store your password safely

Register once — log in next time.

Step 3 — Join Baatryn Öröö club

  2. Search Baatryn Öröö or Баатрын Өрөө
