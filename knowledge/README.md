# knowledge/

Drop extra context here before running the content pipeline. The head-of-research agent reads everything in this folder.

Good candidates:

- GSC query exports (impressions, clicks)
- Google Trends notes
- Wordstat / semantic research
- Founder quotes, club rules, bonus terms
- Competitor notes (background only — do not quote competitor articles)

Treat proprietary club data as **high priority** in research briefs.
