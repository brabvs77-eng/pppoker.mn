# Покерын нэр томьёо — онлайн покер Монгол

**Покерын нэр томьёо** мэдэхгүйгээр ширээнд төөрөх хялбар. Энэ толь бичиг [Baatryn Öröö](https://pppoker.mn/) клубын Hold'em, Omaha кэш тоглоомд хамгийн их ашиглагддаг үгсийг монгол хэлээр тайлбарлана.

## Товч хариулт

Покер = карт + стратеги + мөнгө. Гол нэр томьёо: **блайнд**, **флоп**, **банк**, **колл**, **рейз**, **фолд**, **блеф**, **бай-ин**. Дэлгэрэнгүй доор.

## Ширээ, карт

| Нэр томьёо | Тайлбар |
|-----------|---------|
| Ширээ (table) | Тоглогчид суудаг виртуал стол |
| Блайнд (blind) | Заавал тавих анте — SB жижиг, BB том |
| Банк (pot) | Тоглогчдын тавьсан нийт чип |
| Гарын карт (hole cards) | Зөвхөн танд харагдах нууц карт |
| Борд (board) | Ширээн дээрх нийтлэг 5 карт |
| Флоп (flop) | Эхний 3 нийтлэг карт |
| Тёрн (turn) | 4 дэх нийтлэг карт |
| Ривер (river) | 5 дахь, сүүлийн нийтлэг карт |
| Шоудаун (showdown) | Хэн ялсныг карт харуулж шийдэх |

Hold'em-д 2 гарын карт, Omaha-д 4 — [Hold'em vs Omaha](https://pppoker.mn/articles/holdem-vs-omaha/).

## Үйлдэл (action)

| Нэр томьёо | Тайлбар |
|-----------|---------|
| Чек (check) | Тавих мөнгө байхгүй — дамжуулах |
| Бэт (bet) | Банк руу чип тавих |
| Колл (call) | Өмнөх тавьсан дүнтэй тэнцүү тавих |
| Рейз (raise) | Дүн нэмж өсгөх |
| Фолд (fold) | Гараа хаяж тоглоомоос гарах |
| Ол-ин (all-in) | Бүх чипээ банк руу тавих |

**Фолд** хийх чадвар чухал — бүх гарыг тоглох шаардлагагүй.

## Комбинаци, хүч

| Нэр томьёо | Тайлбар |
|-----------|---------|
| Хос (pair) | Хоёр ижил карт |
| Флэш (flush) | Нэг өнгөний таван карт |
| Стрит (straight) | Дараалсан таван карт |
| Фулл хаус (full house) | Гурав + хос |
| Кикер (kicker) | Ижил комбинацтай үед ялгах карт |

Бүрэн жагсаалт: [покерын комбинациуд](https://pppoker.mn/articles/poker-kombinatsiud/).

## Стратеги

| Нэр томьёо | Тайлбар |
|-----------|---------|
| Позиц (position) | Ширээний байрлал — хожоо илүү давуу |
| Блеф (bluff) | Сул гараар банк хожих оролдлого |
| Тильт (tilt) | Сэтгэл санааны алдагдал — их алдана |
| Флоат (float) | Колл хийж дараагийн улиралд банк авах |
| Рейк (rake) | Клубын шимтгэл — банкнаас суутгана |
| Банкролл (bankroll) | Тоглоомын нийт мөнгөний сан |

Дэлгэрэнгүй: [позицийн хүч](https://pppoker.mn/articles/pozitsiin-huch/), [блефийн урлаг](https://pppoker.mn/articles/blef-uriag/), [банкролл удирдлага](https://pppoker.mn/articles/bankroll-udirdlaga/).

| Нэр томьёо | Тайлбар |

| 9-max / 6-max | Ширээнд хамгийн их тоглогчийн тоо |

## Эхлэгчид хамгийн их андуурдаг

- **Блайнд** = зөвхөн нэг тоглогчийн тавьдаг биш — SB ба BB хоёулаа
- **Флоп** гэдэг нь зөвхөн 1 карт биш — 3 карт нэгэн зэрэг нээгдэнэ
- **Блеф** = бүх гарт хийдэг зүйл биш — зөв мөч, зөв өрсөлдөгчид
- **Ол-ин** хийсэн = дараа нь фолд хийх боломжгүй
