# 存款与提款 — 蒙古在线扑克

**存款**和**提款**是选择在线扑克俱乐部时的首要信任测试。[Baatryn Öröö](https://pppoker.mn/?lang=zh) 支持图格里克 IBAN 存款，符合条件的玩家还可使用 USDT（TRC20）。

| | 金额 / 时间 |
|--|-------------|
| 最低存款 | 10,000₮ |
| 最低提款 | 10,000₮ |
| 存款时间 | 通常 10 分钟 – 2–4 小时 |
| 提款时间 | 约 10 分钟起，最多数小时 |

## 如何存款

1. 安装 [PPPoker 应用](https://pppoker.mn/zh/articles/mongol-poker-xiazai/)
2. 加入 Baatryn Öröö 俱乐部
3. 收银台 → 选择 **IBAN**
4. 按说明转账
5. 仔细核对 IBAN 号码

## USDT（TRC20）

**IBAN 累计存款 100,000₮+** 后可使用 USDT——偏好加密货币时很实用。

## 如何提款

1. 收银台 → 提款
2. 输入 IBAN 账号
3. 最低：10,000₮
4. 等待确认
