# Poker Terms — Online Poker Mongolia Glossary

Without **poker terms**, the table feels confusing. This glossary covers the most used words at [Baatryn Öröö](https://pppoker.mn/?lang=en) Hold'em and Omaha cash games.

## Quick answer

Poker = cards + strategy + money. Core terms: **blind**, **flop**, **pot**, **call**, **raise**, **fold**, **bluff**, **buy-in**. Details below.

## Table & cards

| Term | Meaning |
|------|---------|
| Table | Virtual seat where players sit |
| Blind | Forced bet — SB small, BB big |
| Pot | Total chips in the middle |
| Hole cards | Private cards only you see |
| Board | Five shared community cards |
| Flop | First 3 board cards |
| Turn | 4th board card |
| River | 5th and final board card |
| Showdown | Reveal cards to decide winner |

Hold'em = 2 hole cards; Omaha = 4 — [Hold'em vs Omaha](https://pppoker.mn/en/articles/holdem-vs-omaha/).

## Actions

| Term | Meaning |
|------|---------|
| Check | Pass when no bet to call |
| Bet | Put chips in the pot |
| Call | Match current bet |
| Raise | Increase the bet |
| Fold | Give up the hand |
| All-in | Bet all remaining chips |

## Strategy terms

| Term | Meaning |
|------|---------|
| Bluff | Bet/raise with a weak hand to force folds |
| Position | Where you act relative to the button |
| Kicker | Side card that breaks ties |
| Draw | Hand that needs more cards to improve |
| Nuts | Best possible hand on current board |
| Rake | Small fee taken from pots (club operations) |

## Cash game terms

| Term | Meaning |
|------|---------|
| Cash game | Chips = real money, leave anytime |
| Buy-in | Minimum chips to sit |
| Big blind (BB) | Larger forced blind; stack measured in BB |
| Stack | Your chips at the table |

See [cash games Mongolia](https://pppoker.mn/en/articles/cash-game-mongolia/).
