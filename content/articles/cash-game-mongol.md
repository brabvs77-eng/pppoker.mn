# Кэш тоглоом — онлайн покер Монголд

**Кэш тоглоом** (cash game) нь чип = бодит мөнгө гэсэн үг. Турнир биш — ширээнээс бүтэн цагт гарч, дахин сууж болно. [Baatryn Öröö](https://pppoker.mn/) дээр **онлайн покер** кэш формат 24/7: Hold'em 9-max, Omaha 6-max.

## Товч хариулт

| | Кэш | Турнир |
|--|-----|--------|
| Чип | Мөнгөтэй тэнцүү | Турнирын оноо |
| Гарах | Хэзээ ч | Буцах боломжгүй |
| Блайнд | Тогтмол | Өсдөг |
| Эхлэгчид | Тохиромжтой | Банкролл шаардлагатай |

Монголын тоглогчдын ихэнх нь кэшээр эхэлдэг — 10,000₮-өөр ширээнд суух боломжтой.

## Блайнд гэж юу вэ?

Блайнд = шаардлагатай анте. Baatryn Öröö:

- **250/500** — бага банкролл, суралцах ширээ
- **1K/2K — 10K/20K** — туршлагатай тоглогчид

Жишээ: 250/500 ширээнд 10,000₮ банкролл = 20 big blind — эхлэгчид хангалттай, гэхдээ [банкролл удирдлага](https://pppoker.mn/articles/bankroll-udirdlaga/) баримтлаарай.

## Бай-ин (buy-in)

Ширээнд суухад хамгийн бага чипийн хэмжээ. Ихэвчлэн 20–100 big blind. Хэт бага бай-ин — шатах эрсдэл өндөр; хэт их — нэг ширээнд хэт их мөнгө.

**Зөвлөмж:** Эхлэгчид бага блайнд, 30–40 бай-ин зарчмаар [Hold'em](https://pppoker.mn/articles/holdem-vs-omaha/) эхлэх.

## Кэш vs турнир — аль нь танд?

**Кэш сонго**, хэрэв:
- Уян хатан цаг хүснэ (24/7 ширээ)
- Нэг гар алдсан ч бүх банкролл алдахгүй
- Стратеги аажмаар сурахыг хүснэ

**Турнир сонго**, хэрэв:
- Тогтмол хуваарь, том приз хүснэ
- Банкролл менежмент сайн мэддэг

Baatryn Öröö одоогоор кэш ширээнд төвлөрдөг — [монгол покер систем](https://pppoker.mn/articles/mongol-poker-sistem/) уншина уу.

## Hold'em эсвэл Omaha?

| | Hold'em кэш | Omaha кэш |

| Ширээ | 9-max | 6-max |
| Экшн | Дунд | Өндөр |
| Банк | Дунд | Том |
| Эхлэгч | Тийм | Туршлага хийсний дараа |

Дэлгэрэнгүй: [Hold'em vs Omaha](https://pppoker.mn/articles/holdem-vs-omaha/).
