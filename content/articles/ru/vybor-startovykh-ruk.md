# Выбор стартовых рук — онлайн покер для новичков

**Выбор стартовых рук** — главное решение префлоп в **онлайн покере**. Меньше рук, но сильнее — банкролл живёт дольше. Используйте за Hold'em в [Baatryn Öröö](https://pppoker.mn/?lang=ru).

## Простое правило

Новичкам: **тайтово и агрессивно** — сбрасывайте мусор, рейзите сильные руки.

- Тайт = меньше рук
- Агрессия = рейз, а не лимп с мусором

## Премиум-руки

| Рука | Действие |
|------|----------|
| AA, KK, QQ, JJ | Почти всегда |
| AK, AQ | Большинство позиций |
| TT, 99 | Средняя и поздняя позиция |
| 77, 88 | Только поздняя / баттон |

## Позиция

- **Ранняя** — только AA–KK, AK
- **Средняя** — + QQ, JJ, AQ
- **Баттон** — шире, но без мусора
- **Большой блайнд** — уже вложились — защищайте разумно

## Сбрасывать (новичкам)

- Несвязанные низкие: K2, Q5, J8
- Слабый туз с ранней: A2, K7
- Одномастные коннекторы — не по умолчанию

## Omaha

Четыре карты — больше комбинаций. Сначала Hold'em — [Hold'em vs Omaha](https://pppoker.mn/ru/articles/holdem-vs-omaha/).
