# Покерные термины — словарь онлайн покера Монгол

Без **покерных терминов** за столом легко потеряться. Словарь для Hold'em и Omaha в [Baatryn Öröö](https://pppoker.mn/?lang=ru).

## Коротко

Покер = карты + стратегия + деньги. Базовые слова: **блайнд**, **флоп**, **банк**, **колл**, **рейз**, **фолд**, **блеф**, **бай-ин**.

## Стол и карты

| Термин | Значение |
|--------|----------|
| Стол (table) | Виртуальный стол |
| Блайнд (blind) | Обяз. ставка — SB малый, BB большой |
| Банк (pot) | Общие фишки |
| Карманные карты | Только вы видите |
| Борд (board) | 5 общих карт |
| Флоп (flop) | Первые 3 на борде |
| Тёрн (turn) | 4-я карта |
| Рiver | 5-я карта |
| Шоудаун | Вскрытие карт |

Hold'em — 2 карты; Omaha — 4: [Hold'em vs Omaha](https://pppoker.mn/ru/articles/holdem-vs-omaha/).

## Действия

| Термин | Значение |
|--------|----------|
| Чек (check) | Пропуск без ставки |
| Бет (bet) | Ставка в банк |
| Колл (call) | Уравнять |
| Рейз (raise) | Повысить |
| Фолд (fold) | Сброс |
| Олл-ин (all-in) | Весь стек |

## Стратегия

| Термин | Значение |
|--------|----------|
| Блеф (bluff) | Давление со слабой рукой |
| Позиция | Место относительно баттона |
| Kicker | Старшая боковая карта |
| Дро (draw) | Рука ждёт улучшения |
| Натс (nuts) | Лучшая возможная рука |
| Рейк (rake) | Комиссия клуба с банка |

## Кэш

| Термин | Значение |
|--------|----------|
| Кэш-игра | Фишки = деньги, выход в любой момент |
| Бай-ин | Мин. фишки для входа |
| Big blind (BB) | Большой блайнд; стек в BB |
| Стек | Ваши фишки за столом |

[Кэш-игры Монгол](https://pppoker.mn/ru/articles/kesh-igry-mongoliya/).
